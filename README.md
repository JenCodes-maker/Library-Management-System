# 📚 Library Management System

A full-stack Library Management System built using the MERN Stack (MongoDB, Express.js, React.js, Node.js) to streamline library operations. The application allows librarians to manage books and members, issue and return books, track transaction history, monitor library analytics, and customize application settings.

## 🚀 Features

* 📚 Book Management (Add, View, Update, Delete Books)
* 👥 Member Management (Add, View, Update, Delete Members)
* 📤 Book Issuing System
* 🔄 Book Return System
* 📋 Transaction History Tracking
* 📊 Analytics Dashboard

  * Total Books
  * Total Members
  * Issued Books
  * Returned Books
  * Recent Transactions
* ⚙️ Settings Page

  * Customize Library Name
  * Customize Admin Name
  * Persistent settings using Local Storage
* 🎨 Responsive and Modern User Interface
